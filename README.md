### 基础功能
- **添加新任务**：任务名称、截止日期、详细描述
- **任务状态管理**：标记完成/未完成，支持状态切换
- **任务删除**：移除不再需要的任务
- **本地数据存储**：所有数据保存在设备上，无需联网
- **自动时间记录**：记录任务创建时间

### 展示功能
- **颜色区分**：进行中（蓝色）、已完成（绿色）、已超期（红色）
- **智能筛选**：全部、进行中、已完成、已超期、今日到期
- **智能排序**：按截止日期升序排列（今天/本周/未来/无期限）

### 扩展功能
- **进度追踪**：任务完成度（0-100%）
- **任务备注**：为每个任务添加评论
- **每日总结**：独立的每日总结模块
- **日历视图**：直观的日历展示当前任务


## 设计原则
- **简单易用**：界面简洁，操作直观
- **离线优先**：数据全部本地存储，无需网络
- **专注个人**：不包含社交分享等复杂功能
- **持续优化**：基于使用反馈逐步改进功能# focus
